# Keybored - 键盘防抖工具

修复键盘连击/抖动问题的 Windows 桌面工具，基于 [aardio](https://www.aardio.com/) 开发。

![Screenshot](ScreenShot.png)

## 功能特性

- **键盘抖动屏蔽** — 检测并屏蔽因键盘硬件问题导致的连击（chattering），可自定义阈值（1~2000ms）
- **长按识别** — 自动区分正常长按与抖动连击，长按按键不会被误屏蔽
- **实时日志** — 记录所有按键事件，屏蔽（红色）、长按（蓝色）彩色标注
- **统计面板** — 按键抖动次数、连击间隔统计，按频率排序
- **最小化托盘** — 关闭窗口自动最小化到系统托盘，右键菜单操作
- **钩子自恢复** — 检测键盘钩子失效并自动重新安装，长时间运行稳定

## 下载安装

从 [Releases](https://github.com/jiezg/Keybored/releases) 页面下载最新版 `Keybored.exe`，无需安装，直接运行。

## 使用说明

1. 启动程序，默认抖动阈值为 100ms
2. 在底部输入框中调整阈值（单位毫秒，范围 1~2000）
3. 切换到"日志"标签页查看实时按键记录
4. 切换到"统计"标签页查看各键抖动统计
5. 关闭窗口后程序最小化到托盘，右键托盘图标可恢复界面或退出

### 操作类型说明

| 标注 | 颜色 | 含义 |
|------|------|------|
| 按下 | 默认 | 首次按下，正常放行 |
| 弹起 | 默认 | 按键释放 |
| 长按* | 蓝色 | 长按产生的重复按键，不屏蔽 |
| 屏蔽 | 红色 | 检测到抖动连击，已屏蔽 |

## 技术原理

使用 Windows 低级键盘钩子（`WH_KEYBOARD_LL`）拦截全局键盘消息：

1. **抖动检测** — 记录每个按键的上次按下时间，若两次按下间隔小于阈值则判定为抖动并屏蔽
2. **长按识别** — 跟踪每个键的按下状态，已按下的键再次收到 `WM_KEYDOWN` 时识别为长按重复，跳过抖动检测
3. **钩子健康检测** — 定时发送测试按键探测钩子存活状态，失效时自动重新安装（参考 aardio `key.hotkey` 实现）
4. **异步日志** — 使用 `thread.command` 跨线程异步更新 UI，避免钩子回调阻塞
5. **时间溢出处理** — `KBDLLHOOKSTRUCT.time` 为 `unsigned long` 类型，约 49 天溢出归零，已做跨溢出计算
